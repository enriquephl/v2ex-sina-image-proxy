# v2ex-sina-image-proxy
在 v2ex 中使用百度链接替换失效的微博图床，恢复浏览体验。AI写的。

## How it works
Automatically replace `https?://*.sinaimg.cn/*.(jpg|png|gif)` with `https://image.baidu.com/search/down?url={oringinal sinaimg url}`.
